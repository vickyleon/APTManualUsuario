\chapter{Proceso Buscar un Usuario}
	Este capítulo explica el proceso que se lleva a cabo para buscar
	un determinado usuario en el sistema, y así mostrar los siguientes datos:
	
	\begin{itemize}
	\item Nombre
	\item Institución
	\item Rol
	\item Tipo
	\item Estado
	\item Fecha de Expiración
	\item Teléfono
	\item Correo
	\end{itemize}
